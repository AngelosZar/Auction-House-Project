# Auction-House-Project

Project set up

## Prerequisites

- Node.js and npm installed

## Installation Steps

1. Clone repository
2. Install dependencies:

```bash
# npm install

## Development
# Run on local server
npm run dev

## Building
# Build for production:
npm run build

## Testing
# Run Cypress tests:
# npm run cypress

# Tools
Lint: npm run lint
Fix linting: npm run lint:fix
Format code(prettier): npm run format

```
